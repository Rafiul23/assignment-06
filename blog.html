Discuss the scope od var, let, const.

The scope of variable declared by Var is global or functional, which means we can access the variable from anywhere.On the other hand, the scope of variable declared by let and const is block scope, which means we cannot access the variable from outside a block.


The variable declared by var can be redeclaed and reassigned. But the variable declared by let cannot be redeclared.And we cannot reassign and redeclare a variable declared by const.


The variables declared by var, let and const, are hoisted on the top of their scope. The variables declared by var is initialized with undefined, but in case of let and const, variable ared not initialized.

The use cases of undefined and null

The use cases of undefined are as below:


  1. When we declare a variable without assigning a value, the varibla will be initialized as udefined.For example:

    let name;
    console.log(name); // undefined.

  2. If we call a function without passing necessary number of arguement, the function will set the parameter as undefined.For example:

    let sum = (a,b) => a+b;
    sum(9); // undefined.


  3. When we do not return a value from a function and call the function, it will return undefined. For example:

    function sum (a,b) { let sum = a + b; };
    sum(3, 4); // undefined.

The use cases of null are as below:

  1. Null is an assignment value. It is intentionally assigned by developers. It is assigned, when a variable does not have any value.For example:

    let a = null; // it means at this stage, we don't have any value of a, so we assign the value as null.

  2. If we want to reset a value, we can assign null.For example:

    let newUser = { name: 'Rakib', id: 40 };
    // If in any case, we want to reset the values of newUser object, we can assign null.
    newUser = null;

  3. We can check the presence of something by null which we want find.For example, let's we have a function which will return us a value. We can check the value as follow, 

                        let result = findNumber([...data]);
                    if (result !== null) {
                        return 'data found';
                    } else {
                        return 'data not found';
                    }
